**рЃЌрЃБ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў рЃљрЃа рЃњрЃљрЃЦрЃЋрЃА Python Рђћ рЃћрЃА рЃюрЃЮрЃарЃЏрЃљрЃџрЃБрЃарЃўрЃљ NixOS-рЃќрЃћ.**
рЃљрЃЦ рЃљрЃарЃљрЃцрЃћрЃарЃў РђърЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўРђю рЃљрЃа рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ, рЃАрЃљрЃюрЃљрЃЏ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљрЃерЃў рЃљрЃа рЃЕрЃљрЃгрЃћрЃа.

рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃљрЃарЃўрЃА **рЃАрЃгрЃЮрЃарЃў рЃЊрЃљ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃљрЃарЃЌрЃБрЃџрЃў рЃњрЃќрЃљ** NixOS 25.11+-рЃќрЃћ, **flake-рЃўрЃЌ**.

## 1№ИЈРЃБ рЃ»рЃћрЃа рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћ рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃљрЃа рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ

рЃЌрЃБ рЃЮрЃарЃўрЃЋрЃћ рЃљрЃЏрЃЉрЃЮрЃЉрЃА `command not found` Рєњ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЏрЃљрЃарЃЌрЃџрЃљ рЃАрЃБрЃцрЃЌрЃљрЃљ.

## 2№ИЈРЃБ Python рЃћрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃА **System Layer-рЃА**

Python рЃљрЃарЃўрЃА **рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў**, рЃљрЃарЃљ Home Manager-рЃўрЃА рЃАрЃљрЃЌрЃљрЃЏрЃљрЃерЃЮ.

­ЪЉЅ **Home Manager-рЃерЃў Python рЃљрЃа рЃўрЃгрЃћрЃарЃћрЃЉрЃљ.**

## 3№ИЈРЃБ рЃАрЃгрЃЮрЃарЃў рЃљрЃЊрЃњрЃўрЃџрЃў Python-рЃўрЃАрЃЌрЃЋрЃўрЃА

рЃцрЃљрЃўрЃџрЃў:

```
hosts/laptop/default.nix
```

(рЃљрЃю рЃАрЃљрЃЊрЃљрЃф `environment.systemPackages` рЃњрЃљрЃЦрЃЋрЃА)

### рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ:

```nix

  environment.systemPackages = with pkgs; [
```

­ЪЊї рЃарЃљрЃбрЃЮрЃЏ рЃЮрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃљ?

* рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃЊрЃњрЃћрЃюрЃўрЃАрЃљрЃА 3.12 рЃљрЃарЃўрЃА рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃљ, 3.14 рЃћрЃЦрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃљрЃџрЃБрЃарЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃАрЃўрЃљрЃ«рЃџрЃћрЃћрЃЉрЃўрЃЌ
* рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЊрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЮрЃарЃўрЃЋрЃћ рЃЋрЃћрЃарЃАрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃгрЃЮрЃарЃў рЃљрЃарЃЕрЃћрЃЋрЃљрЃюрЃўрЃљ
* РђъpythonРђю alias-рЃќрЃћ рЃљрЃа рЃЋрЃћрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ Рђћ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃЋрЃўрЃФрЃљрЃ«рЃћрЃЉрЃЌ рЃќрЃБрЃАрЃб рЃЋрЃћрЃарЃАрЃўрЃљрЃА.

## 4№ИЈРЃБ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃфрЃўрЃЎрЃџрЃў (рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ)

рЃљрЃа рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃЮ рЃћрЃА рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў.


рЃЌрЃБ рЃљрЃЦ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљрЃљ Рєњ **evaluation / build / activation рЃцрЃљрЃќрЃљрЃА рЃЋрЃљрЃЊрЃњрЃћрЃюрЃЌ**
рЃћрЃА рЃцрЃљрЃќрЃљ рЃБрЃерЃћрЃфрЃЊрЃЏрЃЮрЃЊ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЋрЃўрЃљрЃарЃЮрЃЌ рЃЊрЃљ рЃўрЃАрЃћ рЃњрЃљрЃЋрЃљрЃњрЃарЃФрЃћрЃџрЃЮрЃЌ.

## 5№ИЈРЃБ рЃњрЃљрЃЊрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ


рЃЌрЃБ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃљ рЃЋрЃћрЃарЃАрЃўрЃљ Рєњ **рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃАрЃгрЃЮрЃарЃљрЃЊрЃљрЃљ рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃў**.

## 6№ИЈРЃБ рЃбрЃўрЃърЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў (рЃарЃЮрЃЏ рЃљрЃдрЃљрЃа рЃЊрЃљрЃЋрЃЉрЃарЃБрЃюрЃЊрЃћрЃЌ)

РЮї `home.packages = [ python ]`

* рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃљрЃЊ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ

РЮї `nix-env -iA nixpkgs.python`

* legacy, рЃљрЃЎрЃарЃФрЃљрЃџрЃБрЃџрЃўрЃљ

РЮї РђърЃарЃљрЃбрЃЮрЃЏ рЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА pipРђю

* NixOS-рЃќрЃћ pip рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃБрЃарЃў рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃбрЃЮрЃарЃў

---

## 7№ИЈРЃБ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉ Python NixOS-рЃќрЃћ (рЃЏрЃЮрЃЎрЃџрЃћрЃЊ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃгрЃЮрЃарЃљрЃЊ)

* Python Рєњ **рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃЮрЃарЃў**
* рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў Рєњ

  * рЃљрЃю `python312Packages.xxx`
  * рЃљрЃю **virtualenv / poetry**
* рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў pip-рЃўрЃЌ рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃљрЃарЃљрЃљ рЃАрЃгрЃЮрЃарЃў рЃњрЃќрЃљ

## ­ЪДа рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃљ (рЃАрЃўрЃЏрЃљрЃарЃЌрЃџрЃћ)

* **Python 3.12** Рєњ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ, рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃњрЃљрЃарЃћрЃЏрЃЮ
* **Python 3.14** Рєњ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ / рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў
* рЃЮрЃарЃўрЃЋрЃћ **рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ**, рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ
* **рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў pyenv**
* **рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃ«рЃћрЃџрЃўрЃЌ рЃњрЃљрЃЊрЃљрЃарЃЌрЃЋрЃљ**

­ЪЉЅ рЃљрЃЏрЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА **Nix**, рЃљрЃарЃљ Python-рЃўрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЏрЃћрЃюрЃћрЃ»рЃћрЃарЃћрЃЉрЃў.

## Рџа№ИЈ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃарЃћрЃљрЃџрЃЮрЃЉрЃљ Python 3.14-рЃќрЃћ

  * рЃљрЃю `python314Packages`
  * **pre-release / dev рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА**

­ЪЉЅ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ
­ЪЉЅ рЃърЃарЃЮрЃЊрЃљрЃЦрЃерЃћрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА Рђћ рЃљрЃарЃљ

## РюЁ рЃАрЃгрЃЮрЃарЃў рЃЏрЃЮрЃЊрЃћрЃџрЃў:

**рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў рЃЮрЃарЃўрЃЋрЃћ рЃЋрЃћрЃарЃАрЃўрЃљ, default-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ**

---

рЃЮрЃарЃўрЃЋрЃћ рЃЋрЃћрЃарЃАрЃўрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃбрЃћрЃЦрЃюрЃџрЃЮрЃњрЃўрЃљ рЃљрЃарЃўрЃА рЃљрЃЏрЃљрЃЋрЃћ рЃЎрЃљрЃбрЃљрЃџрЃЮрЃњрЃерЃў рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ [flake](flake.md), рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃА flake.nix рЃцрЃљрЃўрЃџрЃўрЃЌ рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃЊрЃћрЃЉрЃљ. рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮрЃљ, рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЎрЃў, рЃўрЃЏрЃљрЃЌрЃЌрЃЋрЃўрЃА рЃЋрЃўрЃюрЃф рЃљрЃЦрЃљрЃЏрЃЊрЃћ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃўрЃљ рЃЏрЃўрЃЕрЃЋрЃћрЃБрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃА.

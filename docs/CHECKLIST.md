рЃДрЃЋрЃћрЃџрЃљ рЃърЃБрЃюрЃЦрЃбрЃў рЃљрЃарЃўрЃА **рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃЊрЃљ рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљрЃЊрЃў**.

## ­ЪЪб Flake & Git

- [ ] `flake.nix` рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃЊрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃБрЃарЃўрЃљ
- [ ] `flake.lock` рЃЎрЃЮрЃЏрЃўрЃбрЃерЃўрЃљ (рЃљрЃа рЃљрЃарЃўрЃА рЃгрЃљрЃерЃџрЃўрЃџрЃў)
- [ ] рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃЊрЃљрЃЌрЃљрЃњрЃБрЃџрЃўрЃљ Git-рЃерЃў
- [ ] branch рЃљрЃарЃўрЃА рЃАрЃБрЃцрЃЌрЃљ (рЃљрЃа рЃљрЃарЃўрЃА рЃЊрЃљрЃБрЃЎрЃЮрЃЏрЃўрЃбрЃћрЃЉрЃћрЃџрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў)

рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ:
```bash
git status
````

## ­ЪЊї рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃљ

* рЃћрЃА рЃЕрЃћрЃЦрЃџрЃўрЃАрЃбрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ **рЃДрЃЮрЃЋрЃћрЃџ рЃЊрЃўрЃЊ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљрЃќрЃћ**
* рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃарЃЮрЃфрЃћрЃАрЃерЃў Рђћ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃА рЃњрЃљрЃЊрЃљрЃљрЃбрЃљрЃарЃћрЃЉ рЃљрЃЏрЃљрЃќрЃћ
* рЃљрЃ«рЃљрЃџрЃў host-рЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃАрЃљрЃА Рђћ рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃЉрЃЮрЃџрЃЮрЃЏрЃЊрЃћ
